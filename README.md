# Breast Cancer Classification Analysis

This project performs a comparative analysis of five machine learning classification algorithms using the Scikit-Learn breast cancer dataset. The workflow is built around a centralized pandas DataFrame (`df`) to clean, preprocess, and analyze data efficiently.

## 📊 Project Overview
The primary goal of this project is to classify tumor samples as either **Malignant** or **Benign** based on 30 specific physical measurements. We evaluate five distinct models to determine which mathematical approach provides the highest accuracy.

### Performance Summary
The final evaluation metrics:

| Model | Accuracy |
| :--- | :--- |
| **SVM** | **98.25%** |
| Logistic Regression | 97.37% |
| Random Forest | 96.49% |
| Decision Tree | 94.74% |
| k-NN | 94.74% |

*   **Best Performing Model:** Support Vector Machine (SVM)
*   **Worst Performing Models:** Decision Tree & k-NN (Tied)
